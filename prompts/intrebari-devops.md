# Quiz DevOps — Prompt

Prompt pentru a primi intrebari de interviu DevOps/Cloud la intervale regulate:

```
La fiecare 5 minute vreau sa imi trimiti cate o intrebare de interviu
din domeniul DevOps sau Cloud.
```

## Cum functioneaza

Dupa ce ai echipa configurata (Milchick + Mark + Helly), trimite acest prompt lui Mr. Milchick. Agentii vor incepe sa iti trimita intrebari de interviu la fiecare 5 minute — util pentru pregatire tehnica sau pentru a-ti testa cunostintele.
